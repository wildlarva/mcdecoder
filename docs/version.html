Versioniong policy

Deprecation policy

If some part of the specifications are marked deprecated, they will be removed following the next 2 regular releases.

Versioning policy

<major>.<minor>.<micro>

The number in each part is increased when,

<major>

A significant change is made to the specifications of mcdecoder. 0 means that mcdecoder is under evaluation against a real use case.

<minor>

Some features are added to or removed from mcdecoder.

<micro>

Some bugs are fixed or some improvements are taken place.
